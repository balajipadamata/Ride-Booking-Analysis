# Ride Booking Analysis Dashboard

## Project Overview

This project is an interactive Power BI dashboard built to analyze ride booking data. It provides insights into bookings, revenue, cancellations, customer behavior, and driver performance.

## Objectives

* Analyze total ride bookings
* Monitor completed and cancelled rides
* Track revenue and average ride value
* Identify booking trends over time
* Evaluate customer and driver performance

## Tools Used

* Power BI
* Microsoft Excel
* DAX
* Data Modeling
* Power Query

## Dashboard Features

* Total Bookings KPI
* Total Revenue
* Cancellation Analysis
* Ride Status Overview
* Revenue Trends
* Driver Performance
* Customer Insights
* Interactive Filters and Slicers
